# 决策树模型

决策树模型用于预测客户未来3个月资产是否能提升至100万+。

## 模型特点

- **算法**: 决策树 (Decision Tree)
- **可解释性**: 高
- **主要用途**: 识别客户资产升级的关键决策路径
- **树深度**: 4 (限制深度以提高可解释性)
