# 逻辑回归模型

逻辑回归模型用于预测客户未来3个月资产提升至100万+的概率。

## 模型特点

- **算法**: 逻辑回归 (Logistic Regression)
- **可解释性**: 高
- **主要用途**: 识别客户资产升级潜力

## 文件说明

- `asset_upgrade_predictor.py`: 主要预测脚本
- `coefficients_plot.png`: 逻辑回归系数可视化图
- `逻辑回归解释.md`: 模型系数解释和业务洞察

## 模型性能

- **准确率**: 80.36%
- **AUC**: 87.17%

## 主要特征

1. 月收入 (正向影响最大)
2. 年龄 (正向影响)
3. APP活跃度 (正向影响)
4. 生命周期阶段 (负向影响)
